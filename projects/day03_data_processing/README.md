# Day 3 - 数据处理基础

## 项目说明

这是一个完整的招聘数据分析项目，展示pandas数据处理和SQLite数据库操作。

## 项目结构

```
day03_data_processing/
├── main.py              # 主函数入口
├── utils/               # 工具类模块
│   ├── __init__.py
│   ├── data_loader.py   # 数据加载器
│   ├── data_cleaner.py  # 数据清洗器
│   ├── data_analyzer.py # 数据分析器
│   └── db_manager.py    # 数据库管理器
├── data/                # 原始数据
│   └── jobs.csv         # JD数据
├── output/              # 产出报告
│   ├── analysis_report.txt  # 分析报告
│   └── jobs.db              # SQLite数据库
└── README.md            # 本文件
```

## 功能模块

### 1. 数据加载器 (DataLoader)

- 加载CSV文件
- 显示数据基本信息
- 保存CSV文件

### 2. 数据清洗器 (DataCleaner)

- 薪资字符串解析（"20-35K" → 最小值、最大值、平均值）
- 技能列表提取
- 重复数据移除

### 3. 数据分析器 (DataAnalyzer)

- 薪资统计分析（平均值、中位数、范围）
- 城市分布分析
- 岗位方向分析
- 技能词频统计
- 生成分析报告

### 4. 数据库管理器 (DatabaseManager)

- SQLite数据库连接
- 数据保存到数据库
- SQL查询执行
- 上下文管理器支持

## 运行方式

```bash
python3 main.py
```

## 数据处理流程

1. **加载数据** - 从CSV读取招聘数据
2. **数据清洗** - 解析薪资、提取技能
3. **数据分析** - 统计分析、生成报告
4. **数据存储** - 保存到SQLite数据库

## 学习要点

1. **pandas基础**
   - DataFrame创建和操作
   - 数据读取和保存
   - 数据选择和过滤
   - 数据统计和聚合

2. **数据清洗**
   - 字符串解析
   - 数据类型转换
   - 缺失值处理
   - 重复数据处理

3. **数据分析**
   - 描述性统计
   - 分组统计
   - 词频统计
   - 报告生成

4. **SQLite数据库**
   - 数据库连接
   - 数据存储
   - SQL查询
   - 上下文管理器
