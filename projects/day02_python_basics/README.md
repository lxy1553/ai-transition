# Day 2 - Python基础补齐

## 项目说明

这是一个展示Python基础知识的实战项目，包含数据验证、数据清洗和日志分析功能。

## 项目结构

```
day02_python_basics/
├── main.py              # 主函数入口
├── utils/               # 工具类模块
│   ├── __init__.py
│   ├── log_analyzer.py  # 日志分析器类
│   ├── validator.py     # 数据验证器类
│   └── cleaner.py       # 数据清洗器类
├── examples/            # 示例数据
│   └── app.log          # 示例日志文件
├── output/              # 产出报告
│   └── log_report.txt   # 生成的分析报告
└── README.md            # 本文件
```

## 功能模块

### 1. 数据验证器 (Validator)

提供常用的数据验证功能：
- 邮箱格式验证
- 手机号格式验证
- 非空验证

### 2. 数据清洗器 (DataCleaner)

提供常用的数据清洗功能：
- 文本清洗（去空格、转小写）
- 薪资字符串解析
- 列表去重

### 3. 日志分析器 (LogAnalyzer)

完整的日志分析工具：
- 读取日志文件
- 解析日志格式（时间、级别、消息）
- 统计各级别日志数量
- 提取错误和警告信息
- 生成分析报告

## 运行方式

```bash
python3 main.py
```

## 学习要点

1. **面向对象编程**：使用类封装功能
2. **模块化设计**：工具类分离，职责清晰
3. **静态方法**：@staticmethod装饰器
4. **文件操作**：读取、解析、生成报告
5. **正则表达式**：解析日志格式
6. **数据统计**：Counter计数器

## 输出示例

运行后会：
1. 演示数据验证功能
2. 演示数据清洗功能
3. 分析示例日志文件
4. 在 `output/` 目录生成分析报告
