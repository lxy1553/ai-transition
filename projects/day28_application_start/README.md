# Day 28 - 试投启动包

这个项目用于检查第 4 周结束后是否具备小批量试投条件。

它不会替代真实投递，而是把投递前的材料拆成清单：

- 简历方向；
- RAG 项目 README；
- 演示稳定性；
- 面试问题库；
- 投递记录；
- 项目链接；
- 首批岗位筛选。

## 运行方式

```bash
cd /Users/lxy/Documents/ai_transition
python3 projects/day28_application_start/main.py
```

## 输出文件

```text
projects/day28_application_start/output/application_readiness_results.json
projects/day28_application_start/output/application_readiness_report.md
```

## 生产映射

找工作也要像做项目一样可复盘。
试投前先检查材料是否齐备，小批量投递后再根据反馈调整简历、项目话术和岗位筛选标准。
