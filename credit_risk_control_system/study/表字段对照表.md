### 3.1 dwd_application — 清洗后申请表（18列）

```
扣分规则: 必填缺失-30 | 金额异常-20 | 类型未知-10 | 收入为0-5
及格线: dq_score ≥ 60 通过，< 60 隔离不入DWS
```

```
#  字段名              类型        可空  变化        转换规则
──  ─────────────────  ──────────  ───  ──────────  ────────────────────────────────────────
1   user_id            STRING      ❌   空值填充     空→'MISSING'，扣30分
2   application_id     STRING      ❌   不变         直接映射
3   apply_amount       DOUBLE      ❌   异常修正     负数→0, NULL→0，异常扣20分
4   product_type       STRING      ❌   标准化       cash_loan→CASH_LOAN, None→UNKNOWN,扣10分
5   user_name          STRING      ❌   🛡脱敏       黄敏→黄*（保留首字）
6   id_card            STRING      ❌   🛡脱敏       934184********8691（前6后4）
7   phone              STRING      ❌   🛡脱敏       138****8795（前3后4）
8   occupation         STRING      ❌   标准化       fillna('UNKNOWN')→字典映射
9   monthly_income     DOUBLE      ❌   异常修正     NULL/负数→0，为0扣5分
10  education          STRING      ✅   不变         暂不处理
11  city               STRING      ✅   不变         直接映射
12  ip_address         STRING      ✅   🛡脱敏       生产环境用hash（演示保留原值）
13  device_id          STRING      ✅   不变         直接映射
14  channel            STRING      ❌   标准化       APP_ANDROID/APP_IOS/H5/PARTNER_A/PARTNER_B/UNKNOWN
15  apply_time         TIMESTAMP   ✅   不变         直接映射
16  dq_score        ⭐ INTEGER      ❌   新增         初始100，逐项扣分
17  dq_quarantined   ⭐ BOOLEAN      ❌   新增         dq_score < 60 → TRUE（隔离不入DWS）
18  dt                 STRING      ❌   继承         来自ODS
```
