## 使用说明

今天的目标不是海投，而是做一轮“市场校准”。

做法：

1. 从招聘平台找 10 个和你最接近的岗位
2. 每个岗位只记录最关键的信息
3. 不纠结公司大小，先看岗位要求
4. 重点看这 3 类岗位：
   - RAG / NL2SQL / 数据问答工程师
   - AI 应用工程师
   - 数据工程师（AI方向）

## 建议怎么填

### 方向分类

只填下面三种之一：

- `RAG/NL2SQL`
- `AI应用`
- `数据工程(AI)`

### 关键词

不要整段复制 JD，控制在 5 到 8 个词：

例如：

`Python, SQL, RAG, FastAPI, Docker, Prompt`

### 我的匹配点

写你已经有的东西，例如：

- SQL 强
- 数仓建模经验
- 指标体系经验
- 熟悉业务口径

### 我的缺口

写你现在还不稳的东西，例如：

- RAG 实战不足
- Python 服务化薄弱
- 缺少可展示项目
