## 最终结论（2026-04-23更新）

### 1. 市场最常出现的 5 个技能词

1. **Python** - 基础开发语言，必备
2. **LLM/大模型** - 核心能力，API调用
3. **RAG** - 检索增强生成，高频场景
4. **Prompt Engineering** - 提示词工程，必备技能
5. **SQL** - 数据处理基础（你的优势）

### 2. 最容易切入的岗位类型

**首选：RAG/NL2SQL工程师**
- 原因：结合你的SQL强项和数仓经验
- 匹配度：70%（SQL+数据建模是优势，RAG需补）
- 薪资范围：20-40K

**次选：AI应用工程师**
- 原因：技能要求更广，但门槛相对灵活
- 匹配度：60%（需补Python工程化和LLM实战）
- 薪资范围：18-35K

**保底：数据工程师（AI方向）**
- 原因：最接近当前背景
- 匹配度：80%（数仓经验直接迁移）
- 薪资范围：20-35K

### 3. 未来 2 周最该优先补的 3 个能力

1. **Python 工程化**（第1周）
   - FastAPI框架
   - 模块化开发
   - 配置、日志、异常处理

2. **LLM API 调用**（第2周）
   - OpenAI/Claude API
   - Prompt设计基础
   - 结构化输出

3. **最小可展示项目**（第2周末）
   - SQL解释助手CLI
   - 能跑、能讲、能展示

## 我的最终判断

基于市场调研和当前背景：

- **最容易切入：** `RAG/NL2SQL工程师`
- **第二选择：** `AI应用工程师`
- **保底选择：** `数据工程师（AI方向）`
- **最重要缺口：** `Python 工程化` → `LLM实战` → `RAG项目`
- **最大优势：** `SQL强` + `数据建模` + `业务理解`
- **差异化竞争力：** 数据+AI双背景，不是纯算法也不是纯开发
