# 金融信贷离线仓库为什么要分 ODS、DWD、DWS、ADS？

> 来源: 核心题库 | 分类: 数据仓库

金融信贷离线仓库分层，是为了把原始数据、明细事实、主题汇总和应用指标分开管理。
ODS 保留业务系统原始数据，DWD 负责清洗成统一粒度的明细事实，DWS 按主题和维度提前汇总，
ADS 面向日报、看板和高频业务查询提供稳定指标。

如果不分层，Agent 或数据分析人员会直接面对大量原始表，很容易选错表、混淆口径、扫大表、重复计数，
还可能访问客户手机号、身份证号、银行卡号等敏感字段。
分层后，经营总览优先查 ADS，维度趋势优先查 DWS，授权明细才查 DWD，ODS 默认只用于上游排查。

在 Agent 数据问答里，分层还能降低 NL2SQL 难度。
Schema Catalog 可以告诉 Agent 每张表的层级、粒度、分区、权限和适用问题，
让 Agent 先选对层，再生成 SQL、校验 SQL、执行查询和记录审计。
