# 你会怎么演示一个 NL2SQL 项目？

> 来源: 核心题库 | 分类: NL2SQL

演示 NL2SQL 项目时，不能只展示一个成功生成 SQL 的例子。
我会准备三类样例：正常指标查询、复杂对比或趋势查询、安全阻断查询。
演示顺序是先输入业务问题，再展示问题解析结果，包括指标、维度、时间和风险标记；
然后展示生成的只读 SQL，说明它如何受到 Schema Catalog 约束；
接着展示 SQL Validator 如何检查字段白名单、敏感字段、时间范围和 limit；
然后展示查询执行结果和业务解释；
最后展示被阻断样例，比如导出手机号或缺少时间范围的大表查询，说明系统如何保护权限、隐私和成本。
