# 第一周复盘时，如何把基础能力讲成 AI 应用工程能力？

> 来源: 核心题库 | 分类: 综合

第一周的环境、Python、数据处理、HTTP API、FastAPI 和工程化，
不应该只讲成零散基础课。
面试表达时要把它们串成 AI 应用的底座：
用 Python 做数据处理和业务逻辑，用 HTTP API 接外部服务，
用 FastAPI 暴露能力，用配置、日志和异常处理保证稳定性。
这样就能说明自己不是只会调模型，
而是在补齐从脚本到服务、从本地实验到生产系统的工程能力。
