# 试投前应该检查哪些材料？

> 来源: 核心题库 | 分类: 综合

试投前至少检查简历、项目 README、演示脚本、面试题库和投递记录。
简历里要把金融信贷开发经验和 RAG / NL2SQL 方向连接起来；
项目 README 要说明项目目标、运行方式、核心链路、边界处理和后续优化；
演示脚本要准备成功样例、bad case、citations 和 request_id；
面试题库要能支撑 RAG、SQL、FastAPI、评测、安全和成本问题；
投递记录要记录公司、岗位、渠道、匹配度和反馈，方便复盘。
