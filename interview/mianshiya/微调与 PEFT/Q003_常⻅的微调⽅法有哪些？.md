# 常⻅的微调⽅法有哪些？

> 来源: 面试鸭题库 | 分类: 微调与 PEFT

微调本质上是拿预训练模型在特定任务上做⼆次训练，让模型更适配具体场景。常⻅的⽅法选择得看数据量、算⼒和
任务复杂度。
1）全量微调就是把整个模型的参数都放开训练。效果通常最好，但显存消耗⼤，8 张 A100 都不⼀定扛住 Llama 2 7B
的全量微调。适合数据质量⾼、预算充⾜的场景。
2）LoRA（低秩⾃适应）现在成了主流⽅案。它冻结原模型权重，在旁边加个⼩的低秩矩阵做增量，训练时只更新这
部分。显存能省下 70%，训练速度也快，Hugging Face 上⼤部分开源微调项⽬都在⽤。
# LoRA 核⼼思想： A 下降维度， B 升维，中间做低秩分解
lora_weights = (x @ A) @ B
output = original_output + lora_weights
3）Adapter ⽅法是在 Transformer 层之间插⼊⼩的神经⽹络模块，训练时固定主⼲，只调这些⼩模块。结构改动⽐
LoRA ⼤，但现在⽤得少⼀些。
4）P-Tuning 和 Prompt Tuning 属于提⽰微调，不改模型本⾝，⽽是学⼀组虚拟 token 的 embedding。适合少样本
场景，但效果波动⼤，对初始化敏感。
选哪个得权衡。数据多、资源⾜，可以试全量微调；⼤多数情况下，LoRA 是性价⽐之选，像 Qwen、ChatGLM 的社
区微调基本都靠它打底。
