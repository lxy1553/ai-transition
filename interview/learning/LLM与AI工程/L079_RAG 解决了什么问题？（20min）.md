# RAG 解决了什么问题？（20min）

> 来源: 学习复习计划 | 分类: LLM与AI工程

### 1.1 LLM 的三个"不知道"


```
问题 1: "什么是 night_ops_ratio_30d？"
  LLM 知识截止在训练数据 — 不知道你这个项目的特定概念

问题 2: "截至昨天，各渠道通过率是多少？"
  LLM 不知道实时数据 — 知识库是静态的

问题 3: "user_000042 为什么被拒？"
  LLM 没有企业内部数据的访问权限 — 这是隐私数据

```

### 1.2 RAG 的解决方案


```
用户提问
  │
  ▼
┌─────────────────┐
│  1. 检索阶段      │  ← 从知识库中找出最相关的文档片段
│  向量搜索          │
│  Top-K 检索        │
└────────┬────────┘
         │  "相关文档片段"
         ▼
┌─────────────────┐
│  2. 增强阶段      │  ← 把检索结果 + 用户问题 拼成 Prompt
│  Prompt 构造       │
│  Context 注入      │
└────────┬────────┘
         │  "完整 Prompt"
         ▼
┌─────────────────┐
│  3. 生成阶段      │  ← LLM 基于 Context 回答问题
│  LLM 回答         │
│  引用来源          │
└─────────────────┘

```
