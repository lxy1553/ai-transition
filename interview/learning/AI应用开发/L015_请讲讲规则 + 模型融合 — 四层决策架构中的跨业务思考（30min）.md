# 请讲讲规则 + 模型融合 — 四层决策架构中的跨业务思考（30min）

> 来源: 学习复习计划 | 分类: AI应用开发

### 医疗分诊的融合架构


```
Layer 1 硬规则:
  - 心率=0 → 立即抢救（不用跑模型）
  - 已知过敏药物 → 禁止使用

Layer 2 模型:
  - 疾病风险评分

Layer 3 融合:
  - 模型说低风险 BUT 硬规则说"体温 41°C" → 无视模型，送急诊
  - 模型说高风险 BUT 无硬规则触发 → 送 ICU

Layer 4 策略:
  - ICU / 急诊 / 门诊 / 观察

```
