# 请讲讲特征工程 — 从事件流到特征向量的三种模式中的今日要点

> 来源: 学习复习计划 | 分类: AI应用开发

```
特征工程的三种模式:

模式1 — COUNT WHERE:
  公式: SUM(event_type = X) WHERE event_time IN [ref-N, ref]
  关键: 用 event_time 而非 dt 分区键

模式2 — 比率衍生:
  公式: mean(condition) 或 ratio = A类的SUM / B类的SUM
  关键: 消除用户活跃度偏差

模式3 — 缺失策略:
  公式: fillna(0) 而非 fillna(均值)
  关键: "无数据"本身就是信号

```
