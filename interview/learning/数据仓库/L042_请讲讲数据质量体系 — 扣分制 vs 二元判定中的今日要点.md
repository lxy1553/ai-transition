# 请讲讲数据质量体系 — 扣分制 vs 二元判定中的今日要点

> 来源: 学习复习计划 | 分类: 数据仓库

```
扣分制的三个核心决策:
  1. 扣分阈值: 致命(-30) / 严重(-20) / 轻微(-10) / 提示(-5)
  2. 隔离线: dq_score < 60 → 隔离（一个致命 + 一个严重 + 一个轻微 = 40 < 60）
  3. 质量报告: 不是打印日志，是结构化数据（DQReport）供监控系统消费

扣分权重不是拍脑袋:
  → 需要和下游消费者（AI工程师、BI分析师）沟通
  → 哪些字段缺失"还行"，哪些"绝对不能丢"
```
